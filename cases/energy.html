ИИ-консультант по электроэнергетике

На базе Istok.AI развёрнут отраслевой ассистент: отвечает на вопросы по приказам Минэнерго, постановлениям Правительства и методическим указаниям — с цитатами и ссылками на пункты. Для юристов, тарифных специалистов, сбытовых и договорных подразделений.

Задача клиента

  • Специалисты тратят часы на поиск норм в приказах Минэнерго, постановлениях Правительства и методических указаниях
  • Один вопрос часто требует сверки нескольких документов — вручную легко пропустить связанный пункт
  • Нужны точные ссылки на конкретные пункты для внутренних заключений и переписки с контрагентами
  • Обычный поиск по папкам и PDF не понимает смысл вопроса — только совпадение слов
  • Общедоступные ИИ-чаты не знают отраслевую нормативку и могут уверенно ошибаться

Цели проекта

  • Сократить время поиска нормы с часов до минут
  • Давать ответ только на основе загруженной нормативной базы
  • К каждому выводу — документ, пункт и цитата
  • Обеспечить доступ сотрудникам разных подразделений через единый чат

Решение

RAG-ассистент Istok.AI по отраслевой нормативке

Семантический поиск по векторной базе нормативных актов и генерация ответа на основе найденных фрагментов.

База знаний Нормативные акты с актуальными редакциями: приказы, постановления, методические указания, внутренние регламенты.
Семантический поиск Система находит релевантные фрагменты по смыслу вопроса, а не только по ключевым словам.
Ответ с источником Формулировка + название документа + пункт. Пример: «п. 65(2) Приказа №1178, п. 13 МУ 1554/17».
Контроль эксперта Итоговое решение и оценка рисков остаются за специалистом — система ускоряет поиск, не заменяет эксперта.
Закрытый контур Развёртывание на сервере заказчика, разграничение доступа по ролям. Демо — по приглашению.

Что получил клиент

  • Быстрые ответы по отраслевой нормативке без долгого ручного поиска

  • Единая точка доступа к нормативке для разных подразделений
  • Меньше риска пропустить связанный пункт при сложных вопросах
  • Основа для пилота и масштабирования на смежные базы документов
